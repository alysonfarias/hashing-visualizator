Uniformidade: A probabilidade de um elemento ser mapeado para uma posição da tabela é a mesma para todos os elementos.

Independencia: A probabilidade de um elemento ser mapeado para uma posição da tabela é independente da posição que ele seria mapeado.

Estabilidade: A probabilidade de um elemento ser mapeado para uma posição da tabela é independente dos elementos que já foram mapeados.
